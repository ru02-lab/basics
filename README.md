# basics
just refreshing basic concepts

Beginner:
1.	FizzBuzz – Print numbers 1 to 100, but for multiples of 3 print “Fizz”, for multiples of 5 print “Buzz”, and for multiples of both print “FizzBuzz”.
2.	Palindrome Checker – Check if a given string is a palindrome.
3.	Factorial Calculator – Calculate the factorial of a number.
4.	Prime Number Checker – Check if a number is prime.
5.	Reverse a String – Reverse the characters in a string.
6.	Sum of Array – Calculate the sum of all elements in an array.
7.	Find the Largest Number – Find the largest number in a list.
Intermediate:
1.	Sorting Algorithms – Implement bubble sort, insertion sort, or merge sort.
2.	Binary Search – Implement binary search on a sorted array.
3.	Anagram Checker – Check if two strings are anagrams of each other.
4.	Count Vowels – Count the number of vowels in a string.
5.	Fibonacci Sequence – Generate the first N numbers in the Fibonacci sequence.
6.	Balanced Parentheses – Check if a string of parentheses is balanced.
7.	Merge Two Sorted Lists – Merge two sorted linked lists into one sorted list.
8.	Matrix Transpose – Transpose a given matrix.
9.	Find Duplicates – Find duplicates in an array.
10.	Implement a Stack – Create a stack with push, pop, and peek operations.
